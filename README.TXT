This app is built using Java and Firebase and using the android studio IDE
It is a Social Media app which has features like:
1.Login and Signup
2.Tab layout with 3 tabs (chats, users, profile)
3.Users tab lists all the users of the app using a recycler view
4.The profile tabs contains the username and profile photo which the user can change anytime
5.The user can select anyone they want to chat with from the users tab and start chatting in realtime
6.Once the user starts chatting with someone, the receiver is than added to the chats section so that 
user can view previous chats or chat from there only
7.The chat tab contains features of online/offline
8.The message section contains special features like seen and delivered, and chats are updated in real time.
